ProAim Gear

ProAim Gear é uma loja virtual especializada em produtos de alto desempenho para gamers e profissionais, oferecendo mouses, teclados e mousepads de marcas renomadas. O objetivo deste projeto é criar uma experiência de compra online moderna, com design imersivo e funcionalidades eficientes para o usuário.

A loja oferece um visual escuro, com detalhes em neon para transmitir velocidade, precisão e tecnologia de ponta, criando uma experiência de navegação que se conecta diretamente com o universo gamer.

Tecnologias Utilizadas
Front-End:

HTML5: Estruturação da página.

CSS3: Estilização e design responsivo utilizando Flexbox e Grid.

JavaScript: Interatividade e animações (efeitos de hover, transições e carrinho de compras).

Back-End:

PHP: Para processamento de formulários e funcionalidades de CRUD (criar, ler, atualizar, excluir produtos).

MySQL: Banco de dados para armazenamento dos produtos, categorias e dados do usuário.

Extras:

PHP Sessions: Para gerenciamento do carrinho de compras.

Efeitos e animações com JS: Transições suaves, scroll suave e feedback visual em tempo real.

Responsividade: Design adaptativo para dispositivos móveis.

Funcionalidades
Catálogo de Produtos: Listagem de mouses, teclados e mousepads com filtros por categoria.

Página de Produto: Detalhes dos produtos com imagens, descrição e preço.

Carrinho de Compras: Funcionalidade para adicionar e remover itens do carrinho, com feedback em tempo real.

Administração de Produtos: Painel administrativo simples para adicionar, editar e excluir produtos.

Login Administrativo: Sistema de login para acesso ao painel de administração.

Validação de Dados: Verificação de formulários e segurança no envio de dados.
